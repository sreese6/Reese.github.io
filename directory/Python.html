Work Workout Alert


Code created a sound alert to remind me to workout. Sound file not currently included on the website. I will have a downloadable zip file soon.


import winsound

import time



i = 1

while i <= 10:

#pushups winsound.PlaySound("pushups.wav", winsound.SND_ASYNC) time.sleep(600) #situps winsound.PlaySound("Situp.wav", winsound.SND_ASYNC) time.sleep(600) #lunges or squats winsound.PlaySound("legs.wav", winsound.SND_ASYNC) time.sleep(600) #rep counter print(i) i += 1
